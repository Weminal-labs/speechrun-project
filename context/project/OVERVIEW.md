## What It Is

**SpeechRun** (aka "Let Your Code Talking") is an AI-powered "code podcast" app for the Cloudflare x ElevenLabs hackathon. Users provide a GitHub URL, and two AI agents — **Nova** (PM) and **Aero** (Dev) — analyze the codebase and generate a podcast-style voice conversation about the code. The app has a 3-panel layout with a terminal/ASCII aesthetic: context sidebar (left), audio conversation (center), and mini-app sandbox (right).

## The Problem

Understanding a new codebase is time-consuming and tedious. Developers spend hours reading documentation, scanning file structures, and trying to piece together how things work. Existing code explainers produce dry text summaries that are easy to skim but hard to learn from.

## The Solution

Turn codebase understanding into a listenable experience. Two AI personas — a PM and a Dev — have a natural, opinionated conversation about the code, covering architecture decisions, code quality, patterns, and user impact. The result is a podcast you can listen to while commuting, exercising, or just leaning back from your screen.

## The User

Developers and engineering managers who want to quickly understand an unfamiliar codebase — whether onboarding to a new team, reviewing an open-source project, or evaluating a dependency.

## Differentiator

- Two distinct AI personas having a natural back-and-forth (not a monologue summary)
- Real voice audio via ElevenLabs (not text-to-speech robot voice)
- Structured context generation (Arche-style) alongside the podcast
- Terminal/ASCII aesthetic that feels native to developers

## Success Metrics

- Hackathon demo: paste a GitHub URL, get a playable podcast within 2 minutes
- Audio quality: natural-sounding conversation with distinct voices
- Context quality: generated context is accurate and useful
