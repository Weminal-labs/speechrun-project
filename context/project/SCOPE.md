# Project Scope

> Generated by setup wizard. Updated as priorities shift.

## In Scope — v1 (Hackathon MVP)

- GitHub URL input and repo fetching (public repos)
- Codebase analysis and Arche-style context generation
- Two-agent dialogue generation (Nova PM + Aero Dev)
- ElevenLabs audio generation (per-turn TTS at minimum)
- Audio player with basic controls (play/pause, progress)
- 3-panel layout: context sidebar, conversation center, sandbox right
- Terminal/ASCII aesthetic UI (dark theme, monospace, ASCII art)
- Deploy to Cloudflare Workers
- Durable Objects for stateful agents
- R2 for audio storage
- WebSocket real-time updates

## Out of Scope — v1 (possible later)

- Text to Dialogue API (multi-speaker single audio file)
- Waveform visualization on audio player
- Transcript sync with audio playback (highlight current turn)
- Sandbox mini-app generation
- Voice cloning for custom agent voices
- Multiple podcast episodes per codebase
- Share/embed generated podcasts
- Private repo support (requires GitHub OAuth)
- User accounts and saved history

## Never In Scope

- User authentication or accounts (this is an open demo tool)
- Payment processing
- Mobile native app
- Real-time collaborative editing
- Code execution or sandboxed runtime

## Open Questions

- Exact Workers AI model to use (Llama 3.1 70B vs alternatives)
- ElevenLabs voice IDs for Nova and Aero
- Whether to stream dialogue generation or batch it
